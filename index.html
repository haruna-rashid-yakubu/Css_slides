Introduction

CSS, ou Cascading Style Sheets, est un langage de feuille de style qui permet de contrôler l'apparence des éléments HTML. Il est utilisé pour définir la couleur, la taille, la police, la mise en page et d'autres aspects visuels d'un site Web.

Les Bases du CSS

Une règle CSS est composée d'un sélecteur, d'une propriété et d'une valeur. Le sélecteur permet de désigner l'élément HTML sur lequel appliquer le style. La propriété définit le style à appliquer. La valeur est la valeur du style.

Example


						p {
							color: "green"
						  }

Les Selecteurs en CSS

Les sélecteurs sont utilisés pour désigner les éléments HTML sur lesquels appliquer le style. Il existe de nombreux types de sélecteurs, dont les plus courants sont :

  • Les sélecteurs d'élément;
  • Les sélecteurs de classe
  • Les sélecteurs d'id

Les propriétés CSS

Il existe de nombreuses propriétés CSS qui permettent de contrôler l'apparence des éléments HTML. Voici quelques-unes des propriétés les plus courantes :

  • Couleur
  • Texte
  • Police
  • Mise en Forme
  • Mise en Page

Les valeurs CSS

Les valeurs CSS peuvent être de différents types, tels que des chaînes de caractères, des nombres, des couleurs ou des listes.
